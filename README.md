Aplicativo mobile com navegação (Stack, Drawer ou Tab) para gerenciamento de médicos e usuários.

Funcionalidades:

Cadastro de médicos com nome, especialidade, CRM e foto (capturada pela câmera).
Armazenamento local com SQLite.
Listagem, edição e exclusão de médicos.
Seleção de especialidades via componente (Picker/DropDown) com no mínimo 4 opções.
Cadastro de usuários com nome, CPF, RG e data de nascimento.
Operações de salvar, consultar, editar e excluir usuários no SQLite.
